## Technologies
Project is created with:
* ReactJs : 18.2.0
* React-Router-Dom: 6.3.0
* Node-sass: 7.0.1

## Setup
To run this project, install it locally using npm:

```
$ cd react-flat
$ npm install 
$ npm start
```
